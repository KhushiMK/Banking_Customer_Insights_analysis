# Banking Customer Insights Analysis

This project focuses on analyzing customer financial behavior using a banking dataset to extract meaningful insights and support data-driven decision-making.

## 📊 Project Overview
The analysis explores patterns in loan approval, income distribution, and customer demographics to understand key factors influencing banking decisions.

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- Data Visualization (Matplotlib, Seaborn)
- SQL (basic querying)

## 📈 Key Analysis Performed
- Loan approval distribution analysis
- Income vs loan amount relationship
- Credit history impact on loan approval
- Gender and education-based analysis

## 🔍 Key Insights
- Customers with strong credit history have higher loan approval rates
- Income alone does not guarantee loan approval
- Credit history is the most important factor
- Demographic features show moderate influence

## 📁 Project Structure
- `banking_analysis.py` → main analysis script
- `data.csv` → dataset
- `.png files` → visualizations

## 🚀 Outcome
This project demonstrates practical data analysis skills including data cleaning, exploratory data analysis (EDA), and visualization to derive actionable business insights in the banking domain.
